Project Deploy Process=>

Right click on project

Click on run as -> maven build...

goals -> clean install -> skip test

apply and run-> refresh project

project -> target folder-> copy war file

put war file in D:\softwares\servers\apache-tomcat-9.0.55\webapps

start server D:\softwares\servers\apache-tomcat-9.0.55\bin startup.bat file

once the server starts go to browser localhost:8080/project_name 

close server ctrl+c/shutdown.bat
